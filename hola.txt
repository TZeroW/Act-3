Tutorial para usar esta cosa lolol

Opción A: Usando "Thunder Client" (Recomendado en VS Code)
Es como Postman, pero dentro de tu editor:
Busca en las extensiones de VS Code "Thunder Client" e instálalo.
Haz clic en el icono del rayo que aparecerá a la izquierda.
Haz clic en "New Request".
Cambia GET por POST.
En la URL escribe: http://localhost:3000/register.
Ve a la pestaña Body, selecciona JSON y pega:

{
  "username": "XXXX",
  "password": "XXXXX"
}

Dale a Send. Repite lo mismo para /login para obtener tu Token.

Una vez que tengas el Token del login, para usar GET /tareas en Thunder Client:
Crea una nueva petición GET a http://localhost:3000/tareas.
Ve a la pestaña Auth.
Busca la opción Bearer Token.
Pega ahí el código largo que recibiste.
Dale a Send.


Opción B: Si estás usando Postman, el proceso es muy similar y es la mejor forma de probar APIs profesionales. Aquí tienes los pasos exactos para registrarte, iniciar sesión y usar tu token:

Registrar un usuario (POST /register)
Abre una nueva pestaña en Postman (botón +).
Cambia el método de GET a POST.
En la URL escribe: http://localhost:3000/register.
Haz clic en la pestaña Body (debajo de la URL).
Selecciona el formato raw y, a la derecha, cambia "Text" por JSON.
Pega este código en el cuadro de texto:

{
  "username": "XXXX",
  "password": "XXXXX"
}

Haz clic en Send. Deberías ver el mensaje: "Usuario registrado con éxito"

Cambia la URL a: http://localhost:3000/login (mantén el método en POST).
Asegúrate de que el Body sea el mismo que usaste en el registro.
Haz clic en Send.
En la parte de abajo (Response), verás algo como esto:

{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VybmFtZSI6IlhYWFgiLCJpYXQiOjE3NzAyMzQ1NjcsImV4cCI6MTc3MDIzODE2N30.AbCdEfGhIjKlMnOpQrStUvWxYz1234567890abcdef" 
}

Copia todo ese código largo (el token), lo necesitaremos ahora.
Ver las tareas usando el Token (GET /tareas)
Aquí es donde aplicamos la seguridad que programaste:
Crea una nueva pestaña en Postman.
El método debe ser GET y la URL: http://localhost:3000/tareas.
Ve a la pestaña Authorization (esta está al lado de "Body").
En Type, selecciona Bearer Token.
En el cuadro de la derecha que dice Token, pega el código largo que copiaste en el paso anterior.
Haz clic en Send.

Crear una tarea (POST /tareas)
Para probar que el archivo tareas.json se crea:
Usa el método POST y la URL http://localhost:3000/tareas.
En Authorization, asegúrate de que el Bearer Token siga ahí.
En Body -> raw -> JSON, pega:

{
  "titulo": "Mi primera tarea",
  "descripcion": "Aprender a usar Postman con Node.js"
}

Luego se fijan en le archivo de tareas y veran el JSON con la tarea creada.
